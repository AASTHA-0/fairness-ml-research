# Fairness in Machine Learning – COMPAS Dataset Analysis

This project analyzes potential bias in the COMPAS recidivism prediction dataset.

## Dataset
COMPAS Scores dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Goal
To explore racial bias and fairness issues in machine learning predictions.

## Files
fairsight_research.ipynb – Jupyter notebook with analysis  
compas_scores.csv – dataset used for analysis
